# Social Media & Mental Health Dashboard (Power BI)

## Project Overview
This dashboard explores the relationship between social media usage and mental health risk using Power BI.

## Tools Used
Power BI, Power Query, DAX, Excel

## Dashboard Pages
- Executive Summary
- Screen Time Distribution
- Mental Health Risk by Age Group
- Gender Comparison

## Key Insights
- Higher screen time is associated with higher mental health risk
- Having social media detox days are helpful
